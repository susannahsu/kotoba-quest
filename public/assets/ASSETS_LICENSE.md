# Asset Licensing

## Current state

The game currently ships **no image or audio files**. All visuals are **original placeholder
geometry** drawn procedurally at runtime in `src/game/scenes/BootScene.ts` (simple shapes for
tiles, characters, and enemies). Pronunciation uses the browser's **Web Speech API** (no audio
files bundled).

**Nothing here is derived from the *Mushoku Tensei* anime or novels.**

## When real art/audio is added

Replacement art and audio must be **original** or **CC0 / royalty-free**, and every pack must
be credited in this file with its source and license. Good CC0 sources:

- Kenney game assets — https://kenney.nl/ (CC0)
- Liberated Pixel Cup (LPC) sprites — note their CC-BY-SA / GPL terms and attribute properly
- itch.io CC0 RPG tilesets, UI, and SFX packs

Do **not** add ripped or copyrighted assets to this repository.
